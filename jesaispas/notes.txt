Format VCF 
entête explique les diff etapes fait pour l'optenir 
contig = seq de ref
ALT = dif types de variation
...
CHROM = pour nous toujours 1
POS = position de la mutation
ID = de la mutation
REF = les seqeunce de la reference a cette position
ALT = la seqeunce muté
QUAL = qualité
...
individus: ex :P30-5.trimed1000 : nom de lindifidu (on a enleve toutes les seq de moin de 1000 pour le P30 sous echantillon 5)
peut avoir plusieur individus dans un meme VCF, on aura PASS si mutation presente dans l'indifidu et . si pas presente
DV = mutation presente dans 
DR = mutation pas presente dans 
AF = frequence allelique

sous forme graph et sous forme exel 

bedgraph : fichier qui permet de voir couverture des l'ens des seq sur le genome
.stats : info de comment les reads sont mappé sur la reférence 

Pour un VCF on a du rouge quand on a une AF de 1 (homozygote) et bleu quand >1 (heterogygote) avec seuil de 0.1
